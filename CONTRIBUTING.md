# Contributing to IraqQuant

نرحب بمساهماتك! 🎉

## كيف تساهم

1. Fork المشروع
2. أنشئ branch للميزة الجديدة (`git checkout -b feature/amazing-feature`)
3. Commit التغييرات (`git commit -m 'Add amazing feature'`)
4. Push للـ branch (`git push origin feature/amazing-feature`)
5. افتح Pull Request

## قواعد الكود

- اتبع PEP 8 للكود Python
- اكتب docstrings لكل الدوال
- أضف tests للميزات الجديدة
- حافظ على التوثيق محدث

## الإبلاغ عن المشاكل

استخدم GitHub Issues للإبلاغ عن:
- Bugs
- طلبات ميزات جديدة
- أسئلة عن الاستخدام

شكراً لمساهمتك! 🙏
